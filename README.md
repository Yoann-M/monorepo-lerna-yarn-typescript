# Monorepo-lerna-yarn-typescript
> Typescript Monorepo managed with Lerna and Yarn Workspaces

## Used versions :
- Lerna v3
- Yarn v1
- TypeScript v4
- Node v12 (needed by Stencil.js)

## Install project :
```bash
yarn install
```
## Build project :
```bash
yarn build
```
## Build and watch project :
```bash
yarn build:watch
```
